# layui-admin

## 针对tp5.1开发的，layui的后台模板。便捷的使用方式，快捷的开发方式，希望能够对你有帮助.

### 通过代码极速创建数据表格、表单

## 版本:1.0 beta

## 说明

测试版，未正式上线。请慎重使用。

文档未完善，readme 简单看下，或者看代码。


## 安装方式

beta版本安装

```shell
$ composer rquire thans/layui-admin:1.*@beta
```

## 安装完成后初始化

```php
php think layuiAdmin:install

```
